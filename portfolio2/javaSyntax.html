Java Syntax Lab: Balloons

Using a processing language based off java, I transormed a tractor race into a balloon flying animation. In this lab i learned how modifying parts of code can provide a completly different outcome than you started with and it gave me a bit of practice with Java. Below the animation is its code if you are interested. Refresh to enjoy the animation again!

Restart the animation by refreshing the browser page.

Resources Used: Processingjs, Open Processing, Htmlify

THE CODE

float speedR, speedG, speedB, speedY, speedP;
void setup() { //runs once on start
 size(600, 300);
 speedR=random(.8,1.2);
 speedG=random(.8,1.2);
 speedB=random(.8,1.2);
 speedY=random(.8,1.2);
 speedP=random(.8,1.2);
}
void draw() { //Loop Continuously
 background(#CCCCCC);
 drawBalloon(#FF0000,50,speedR);
 drawBalloon(#00FF00,100,speedG);
 drawBalloon(#0000FF,200,speedB);
 drawBalloon(#FCFF20,300,speedY);
 drawBalloon(#AF1BB1,400,speedP);
}
void drawBalloon(color c, int xStart, float speed){ //runs when called
 resetMatrix();
 translate(xStart, -frameCount*speed, xStart);
 if (xStart, frameCount*speed >= 600-90) { // stops loop for photo finish when winner finishes
 noLoop();
 }
 fill(c);
 line(150, 310, 150, 400);
 ellipse(150, 300, 50, 50); }
